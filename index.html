Move To MifosX

A migration tool to migrate data from Mifos 2.x to MifosX

View project onGitHub

Data Migration Tool.

This tool will help MFI's who are using Mifos and wants to move to MifosX.

  • Migration of Centers, Groups, clients entity wise
  • Client Family details
  • Migration of Loans – both individual client and group loans
  • Support for Flat Rate of interest type
  • Support for Declining Balance interest rate type
  • Migration of Individual clients i.e. clients not associated with any groups or Center
  • Migration of groups independent of centers
  • Migration of Additional fields – Questions groups
  • Validations for
  • Mandatory field in MIFOS
  • Duplicate records check
  • Range checks on the loan product offerings
  • Hierarchy check to validate the loan officer -> Center -> Group->Client link
  • Loan Product Applicability : Products applicable to groups should have loans created for the groups only and not clients
  • Loan creation with product frequency Mismatch : Validation such that the recurrence for the loan product matches the customer/center schedule
  • Date Validations :
  • Loan creation date should be greater than center creation date ( or MFI joining date)
  • Group MFI joining date should be greater than center creation date
  • Disbursement date in loan account cannot be less than loan creation date
  • Should not allow future dated values for MFI joining date, loan creation date etc
  • Migrating One Time fees
  • Migrating Group loans
